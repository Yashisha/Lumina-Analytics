# ◈ Lumina Analytics

> *Your metrics. Crystal clear.*

A clean, modern SaaS analytics dashboard built with **React** and **Recharts** — featuring revenue tracking, user growth, churn rate, traffic sources, and real-time activity feed.

---

## ✨ Features

- 📊 Revenue & user growth area charts
- 📉 Monthly churn rate bar chart
- 🌐 Traffic source breakdown
- 🏆 Top pages by views
- 🔔 Recent user activity feed
- 💰 Animated KPI stat cards
- 🎨 Clean light UI with DM Sans + DM Mono fonts

---

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/Lumina-Analytics.git
cd Lumina-Analytics
```

### 2. Install dependencies
```bash
npm install
npm install recharts
```

### 3. Run locally
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
